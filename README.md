# us-insurance-costs-cc
This is a high-level analysis of US insurance costs. It seeks to get an initial impression of the effects that different life factors have on the cost of insurance by visualising insurance costs in Python.

This project comes from part 6 of the Codecademy course "Data Scientist: Natural Language Processing Specialist" which you can find here: https://www.codecademy.com/learn/paths/data-science-nlp .
